# Operation Guesser 

The operation guesser receives a sequence of 
numbers, a target value, and returns a string
representing the arithmetic operations required
to form the target.

## Examples

```
>>> solve((1, 2, 3, 4), 10)
1 + 2 + 3 + 4
>>> solve((10, 20, 5), 6)
(20 + 10) / 5
```
